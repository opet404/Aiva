BAB 4: STUDI KASUS OMEGA RANSOMWARE (Hari 45-52)

LATAR BELAKANG

Pada hari ke-45, dunia dikejutkan dengan serangan ransomware yang menargetkan 50 perusahaan Fortune 500 di Eropa dan Asia. Ransomware ini bernama "Omega" karena menggunakan simbol omega (Ω) dalam proses enkripsinya. Dalam 48 jam pertama, 10.000+ sistem terinfeksi, dan para korban menerima tuntutan tebusan sebesar $50.000 dalam Bitcoin, dengan tenggat waktu 72 jam.

ANALISIS STATIC OMEGA

Dr. Sarah Chen dan Tim Alpha segera memulai analisis:

Binary Properties:
- Ukuran file: 2.3 MB
- Packer: UPX (terdeteksi)
- Architecture: x86-64
- OS: Windows 10/11, Windows Server 2016/2019/2022
- Compiler: Visual C++ 2022
- Compilation date: 15 Januari 2023

Section Analysis:
- .text: 1.2 MB (executable code)
- .data: 500 KB (initialized data)
- .rdata: 400 KB (read-only data)
- .rsrc: 200 KB (resources)

Import Analysis:
- kernel32.dll: 45 functions (CreateFile, ReadFile, WriteFile, DeleteFile, GetFileAttributes)
- advapi32.dll: 23 functions (RegCreateKey, RegSetValue, RegDeleteKey)
- user32.dll: 12 functions (MessageBox, SetWallpaper, SystemParametersInfo)
- ws2_32.dll: 18 functions (networking)
- crypt32.dll: 15 functions (cryptography)
- ntdll.dll: 8 functions (low-level)

String Analysis (Partial):
- "Omega Ransomware v2.3"
- "Your files are encrypted with Omega"
- "Pay $50,000 in Bitcoin to recover"
- "OMEGA_READ_ME.txt"
- "C:\Users\Public\Documents\OMEGA_NOTICE.html"
- "HKEY_CURRENT_USER\Software\Omega"
- "C:\Windows\System32\drivers\Omega.sys"
- "http://omega-c2-{random}.onion"
- "https://omega-payment-{random}.onion"

Obfuscation Technique:
- 7 lapisan packing dengan UPX (3x)
